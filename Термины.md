# Термины и определения

Этот файл содержит список терминов, используемых в регламенте FedMash, для которых необходимо дать определения.

## Термины соревнований

### Основные понятия
- <a id="судья"></a> **Судья** - это официальное лицо, обеспечивающее подсчёт очков, соблюдение регламента, правил безопасности и честности соревнований.
- <a id="су"></a> **СУ (Спец Участок)** - специальный участок трассы для прохождения на модели внедорожника.
- <a id="ворота"></a> **Ворота** - препятствие, состоящее из двух вешек с указанием направления прохода между ними.
- <a id="бонусные-ворота"></a> **Бонусные ворота** - дополнительные (не обязательные) ворота на СУ, дающие больше очков.
- <a id="вешка"></a> **Вешка** - маркер, обозначающий границы ворот.
- <a id="посох-судьи"></a> **Посох судьи** - инструмент судьи для крепления лебедки.
- <a id="техническая-комиссия"></a> **Техническая комиссия** - проверка модели перед соревнованиями на соответствие заявленному классу.
- <a id="финальная-техническая-комиссия"></a> **Финальная техническая комиссия** - проверка модели после прохождения всех СУ для начисления бонусных очков.
- <a id="карточка-участника"></a> **Карточка участника** - документ, выдаваемый после технической комиссии.
- <a id="дисквалификация"></a> **Дисквалификация** - удаление участника с соревнования за ненормативное поведение на мероприятии.


### Термины оценки
- <a id="Очки-модели"></a> **Очки модели** -  очки, начисляющиеся за соответствие модели определённым требовариям; очки складываются из базовых и бонусных очков.
- <a id="базовые-очки"></a> **Базовые очки модели** - обязательные очки за соответствие требованиям класса.
- <a id="бонусные-очки"></a> **Бонусные очки модели** - дополнительные очки за детализацию и аксессуары.
- <a id="очки-за-су"></a> **Очки за СУ** - очки, начисляемые за прохождение специальных участков.
- <a id="время-прохождения-су"></a> **Время прохождения СУ** - время, затраченное участником на прохождение СУ.
- <a id="незавершение-су"></a> **Незавершение СУ** - не прохождение всех нумерованных ворот в отведенное время.



##  Технические термины

### Общие технические термины

- <a id="колёсная-база"></a> **Колёсная база** - расстояние между первой и последней осями модели.
- <a id="масштаб"></a> **Масштаб** - соотношение размера модели к оригиналу (1/10, 1/8); определяется по колёсной базе, при этом длина и ширина должны соответствовать выбранному масштабу с допуском $\pm 10\%$.
- <a id="внешний-диаметр-колес"></a> **Внешний диаметр колес** - общий диаметр колеса с покрышкой.


### Шасси
- <a id="рама"></a> **Рама** - несущая конструкция шасси, состоящая из двух продольных рельс и нескольких перемычек, жёстко скрепляющих рельсы рамы между собой.
- <a id="скид"></a> **Скид (skid)** - площака под трансмиссию или раздаточную коробку.
- <a id="утяжелители"></a> **Утяжелители** - дополнительные грузы для улучшения развесовки.



### Опции трансмиссии 

- <a id="4ws"></a> **4WS** - система четырехколесного руления (Four Wheel Steering).
- <a id="dig-юнит"></a> **DIG-юнит** - устройство для блокировки заднего моста.
- <a id="овердрайв"></a> **Овердрайв** - система увеличения передаточного числа переднего моста.



### Кузов
- <a id="кузов"></a> **Кузов** - это его
внешняя оболочка и основной конструктивный элемент, предназначенный для размещения водителя, пассажиров и груза. Он обеспечивает жесткость конструкции, защищает от внешних воздействий, формирует внешний облик и определяет функциональность машины.
- <a id="хардбоди"></a> **Хардбоди (hardbody)** кузов - жесткий кузов из пластика или металла.
- <a id="лексановый"></a> **Лексановый** кузов - кузов, выполненный из бутылочного пластика или поликарбоната.
- <a id="кузов-клетка"></a> **Кузов-клетка** (пространственная рама) - это жесткая несущая конструкция, состоящая из труб и создающая форму автомобиля.
- <a id="лифт-кузова"></a> **Лифт кузова** - поднятие кузова относительно шасси.
- <a id="фендеры"></a> **Фендеры** - расширители арок колес.
- <a id="подкрылки"></a> **Подкрылки** - защитные дугообразные кожухи из пластика, полиэтилена или металла, устанавливаемые внутри колесных арок автомобиля и отделяющие колёсное пространство от внутреннего пространства авто.
- <a id="водитель"></a> **Водитель** - пластиковая фигурка человека в салоне, выполненная в масштабе модели.
- <a id="штурман"></a> **Штурман** - пластиковая фигурка человека в салоне, выполненная в масштабе модели.


### Бампера и пороги
- <a id="бампер"></a> **Бампер** - элемент обвеса кузова, защищающий кузов от повреждений; бампер должен выступать вперёд кузова по все своей ширине, он может быть либо силовым (тое. жёстко закреплённым на раме), либо интегрированным с Хардбоди кузовом.
- <a id="трубчатые-бампера"></a> **Трубчатые бампера** - бампера из труб.
- <a id="бампера-из-профиля"></a> **Бампера из профиля** - самодельные бампера из металлического профиля.
- <a id="пороги"></a> **Пороги** - это конструктивные элементы кузова, расположенные в нижней части по бокам между передними и задними крыльями. Они обеспечивают жесткость, прочность кузова.
- <a id="слайдеры"></a> **Слайдеры** - это силовые пороги или защитные балки, закрепленные непосредственно к раме внедорожника. Они защищают нижние панели кузова, пороги и двери от повреждений при езде по камням, пням и бездорожью.



### Дополнительное оборудование
- <a id="лебёдка"></a> **Лебёдка** - это
тяговый механизм, устанавливаемый на внедорожник для самовытаскивания или буксировки.
- <a id="копийная-лебёдка"></a> **Копийная лебёдка** - функциональная лебедка, имитирующая настоящую, и устанавливающаяся на передний силовой бампер модели.
- <a id="серво-лебедка"></a> **Серво-лебедка** - лебедка с сервоприводом и прикреплённой к нему катушкой; предполагает скрытую установку на шасси.
- <a id="динамический-трос"></a> **Динамический трос** - лебедка на основе резинового жгута.
- <a id="сенд-траки"></a> **Сенд-траки** - специальные прочные пластины (металлические, пластиковые или композитные) с перфорацией и грунтозацепами, предназначенные для преодоления бездорожья.
- <a id="запасное-колесо"></a> **Запасное колесо (запаска)** - дополнительное колесо на модели; с точки зрения регламента считается третим лигитимным сенд-траком.
